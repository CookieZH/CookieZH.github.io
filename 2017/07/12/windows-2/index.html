接下来，只生成文件，生成之后，可以在Debug目录里找到.exe文件。打开
cmd，到Debug目录下，进行如下操作：
cd vc++\touch\Debug //这是我的路径
dir //查看当前目录下的文件
touch //执行touch
touch 1.txt 2.txt //创建文件1.txt,2.txt
此时，文件也是创建在当前目录Debug下的，如果想创建在别的地方，需要修改路径。

再次dir,我们可以看到目录里多了文件1.txt,2.txt，说明创建成功！
